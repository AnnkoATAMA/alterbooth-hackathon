# alterbooth-hackathon

---

## Project Description
このプロジェクトはAlterboothのハッカソンに参加するためのリポジトリです。

## プロジェクトの目的
ゲーミフィケーション目標管理
### ターゲット層
社会人、学生、先生(現職が管理職、開発者、デザイナー、学生、教員)

## プロジェクトの機能
グループの機能、タスクをツリー化し管理できるアプリ。

## プロジェクトの構成
### frontend

- html
- css
- javascript

### backend

- python(fastAPI)
- DB: MySQL
- ORM: SQLAlchemy
